ВЗАИМОДЕЙСТВИЯ ОБЪЕКТОВ

1. Push / Толкать

Происходит при совершении хода на место, занимаемое каким-либо другим объектом.
Объект перемещается, в случае, если за ним есть свободное пространство или расположен другой объект, который может каким-либо образом освободить занимаемое пространство. При этом движущийся объект занимет освободившееся место.

2. Pull / Тянуть

Если при совершении хода непосредсвенно вплотную к объекту в направлении противоположном направлению хода находиться объект который можно тянуть, то объект перемещается за тянущим его.

3. Join / Присоединить

При совершении хода на место, занимаемое каким либо объектом, этот объект будет присоединён к движущемуся. Перемещение не будет совершено.

4. Slap / Прилепить

При совершении хода на место, занимаемое объектом, движущийся объект станет его частью. Перемещение не будет совершено.

5. Destroy / Уничтожить

При перемещении объект цель уничтожается. Перемещение совершается на освободившееся место.

6. Die / Умереть

При столновении с объектом, от которого можно умереть, движущийся объект уничтожается

В случае отсутствия взаимодействия между объектами действие просто не совершается.
